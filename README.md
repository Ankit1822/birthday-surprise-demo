# 🎂 Birthday Surprise Web App

An interactive full-stack birthday surprise web application built with HTML, CSS, JavaScript, Node.js, and Express.

## ✨ Features

- 🔐 Name Verification
- 🖼️ Protected Image Gallery
- 🎵 Protected Background Music
- 🎁 Interactive Gift Animation
- 🎂 Birthday Countdown
- ❤️ Heart Catch Game
- 🧩 Heart Puzzle
- 🌹 Bouquet Animation
- 🎰 Lucky Wheel
- 💌 Secret Letter
- 📱 Responsive Design

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Node.js
- Express.js
- Express Session

## 🚀 Installation

```bash
git clone <repository-url>
cd birthday-surprise-demo
npm install
npm start
```

Open:

```
http://localhost:3000
```

## 📁 Note

This is a demo version of the project.
Personal photos, music, and names have been replaced with sample assets to protect privacy.
